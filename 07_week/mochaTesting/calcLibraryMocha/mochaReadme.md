# Mocha testing

## links

-   home: https://mochajs.org/
-   chai library: https://www.chaijs.com/
-   chai language chains: https://www.chaijs.com/api/bdd/

## Install deps and create test dir

```shell
npm i mocha chai --save-dev
```

add script to the package.json

```json
"scripts": {
    "test": "mocha"
}
```

create test dir

```sh
mkdir test
```
